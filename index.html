Projetos

API-DSM-1SEMESTRE-2025

Desenvolvemos uma plataforma web com dados sobre o desempenho dos municípios do Estado de São Paulo no comércio exterior, com base em informações públicas do Ministério do Desenvolvimento. A proposta é fornecer dados claros e acessíveis para auxiliar na tomada de decisões. Utilizamos pandas no Google Colab para tratar arquivos .csv e Flask para criar o site. O projeto superou desafios como o processamento de grandes volumes de dados e a integração entre análise e visualização.

O projeto foi desenvolvido usando o metodo ágil SCRUM, com reuniões semanais para acompanhamento do progresso. A Nossa equipe é composta por 8 integrantes, incluindo um Product Owner e um Scrum Master. O projeto foi dividido em 3 sprints de 3 semanas, com entregas parciais a cada sprint. Reuniões diárias foram realizadas para alinhar as atividades e resolver impedimentos.

Meu papel esta sendo o de desenvolver o back-end da aplicação, utilizando pandas e Python, para limpar os arquivos de exportações e importações e criar filtros e gráficos no Colab. Na primeira sprint do projeto eu atuei como Srum Master, porém na segunda sprint eu passei a atuar como Product Owner.

O projeto está em andamento e a previsão de entrega é para o final do semestre (05/2025).
